Como é uma estrutura básica de um HTML??

(Voltar)

O documento HTML abre com varios tipos de editor de texto ou um programa específico.

O navegador faz a leitura para que esse conteudo seja visualizado pelo :

  • google chrome
  • Safari
  • Mozilla Firefox

O site geralmente são compostos por varias paginas em HTML, exemplo:

  • Homepage
  • Página de contato
  • Página de produtos
"Documentos HTML são arquivos que terminam com uma extensão .html ou .htm. Um navegador lê o arquivo HTML e renderiza o seu conteúdo para que os usuários da internet possam vê-lo. Todas as páginas HTML possuem uma série de elementos, que consistem num conjunto de tags e atributos. Os elementos HTML são os tijolos de construção de uma página da internet. Uma tag diz para o navegador onde um elemento começa e termina, enquanto um atributo descreve as características de um elemento."

Entendendo a diferença entre listas ordenadas e não ordenadas

(Voltar)

Lista ordernada precisa da Tag filha e os marcadores são com numero.Exemplo:

<ul>
<li > Homepage </li > </ul >

Lista não ordenada Precisa da tag filha e os marcadores são com bolinhas.Exemplo:

<ol>
<li > Homepage </li > </ul >
